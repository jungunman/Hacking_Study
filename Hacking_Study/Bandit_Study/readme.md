# 훈련 과정

## 어떻게 풀어나가려고 노력했는가?
- Bandit에서 주어진 명령어들을 검색하여, 개념과 사용법을 알고 적용하려고 노력했습니다. 
답을 찾아가는 과정 자체를 이해하기 어려웠다면, 구글 번역기나 먼저 경험해본 선배들의 지문 해석을 통하여 이해하려고 노력했습니다.
그래도 답을 찾지 못했다면 답안을 참고했습니다.
답안을 참고한 문제들은 복습을 통하여 개념과 사용하는 이유, 원리에 대해서 이해하려고 노력했습니다.

## 풀이 과정
- 풀이 과정을 쓴 이유는, 문제에 접근할 때 어떻게 접근했고 무슨 시행착오를 겪었는지 기억하기 위함입니다.
선배님들의 답지에는 요점과 명령어만 딱딱 써져있어. 왜? 이렇게 문제에 접근했는 지 이해하기 어려워서 저는 풀이 과정도 남깁니다.
